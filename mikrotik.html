: : : CARA SETING MIKROTIK

1. Bagaimana cara terhubung Ke Hotspot?
  1. Silakan beli voucher disini
  2. Aktifkan wifi perangkat anda, hubungkan ke SSID kami
  3. Setelah terhubung buka browser anda lalu ketik http://mks.net/
    4.
2. Halaman Login Tidak Bisa Di Buka?
  1. Silakan hapus cahce browser anda
  2. Buka lagi URL login http://mks.net/
  3. Masukan kode voucher/username login anda
  4. Jika masih mengalami kesulitan silakan hub admin
    5.
3. Bagaiamana Cara Cek Sisa Masa Aktif?
  1. Pastikan anda sudah login ke hotspot kami
  2. Buka browser, lalu ketikan http://mks.net/status
  3. Status voucher anda akan terlihat di halaman ini
    4.
4. Sudah Login Tapi Internet Tidak Jalan?
  1. Hubungi admin, laporkan kendala anda
  2. Kami akan melakukan pengecekan
  3. Jika masalah dari server, maka akan kami berikan voucher pengganti
    4.
5. Bagaimana Jika Ingin Berhenti Internetan?
  1. Anda harus logout agar masa aktif tidak terus berjalan jika ingin STOP internetan
  2. Silakan buka browser lalu ketikan http://mks.net/status kemudian klik logout
  3. Jika ingin login kembali silakan ikuti langkah ke 1
    4.
